# Knowledge Management App: API Server

## Getting Started

Ensure that the Docker Engine is running before proceeding.

```console
# /Makefile

make api
```

## Technologies and Tooling

- Runtime: .NET 10.0 SDK
- Framework: ASP.NET Core
- Logging: Serilog
- API Documentation: OpenAPI / NSwag
- ORM: Entity Framework Core
- Database: PostgreSQL
- Mapping: Mapperly
